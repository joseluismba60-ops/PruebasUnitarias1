Proyecto basico de Java enfocado en la creacion de pruebas unitarias con JUnit 5 para validar datos de un registro.
El proyecto contiene pruebas unitarias para una clase llamada ValidadorRegistro donde se validan distintos campos comunes: nombre, contraseña, email y edad.
El objetivo es practicar como escribir tests correctamente y validar distintos escenarios (validos e invalidos).
